Install

# Whole suite (recommended — preserves cross-skill routing):
git clone https://github.com/oliverschmidtprietz/EU-AI-Act-Suite.git
cp -r EU-AI-Act-Suite/ai-act-* ~/.claude/skills/

# Single skill (standalone):
cp -r EU-AI-Act-Suite/ai-act-classifier ~/.claude/skills/

Some skills cross-reference siblings (e.g. ai-act-obligations reads a timeline file from ai-act-high-risk). Installing a single skill solo loses that cross-routing; the skill still works with reduced linking. Install the whole suite for full behaviour.
